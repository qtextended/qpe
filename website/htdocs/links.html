Applications for Qt Palmtop

These applications have been specifically ported to the Qt Palmtop environment.
  • SID Player - player for SID music files.
  • Konqueror - the web browser from KDE.
  • QPECIN - chinese input method.
  • NetHack - the classic RPG goes mobile.
  • Synchronization Framework - in production at Trolltech [no website yet].
  • Desktop-side Tools - in production at Trolltech [no website yet]. Multi-platform.
  • HTML Editor - in production at Trolltech [no website yet].

Qt-related links

Qt Palmtop is written with the Qt API, using Qt/Embedded. The Qt API is an emerging standard for cross-platform GUI development - the same API also lets you compile on Windows, X11, and MacOS.

Many projects use Qt, far to many to list here. But here are just a few:

  • The KDE Project - entire desktop environment written on top of Qt.
  • Free software and contributions - a large list of Qt-based software.
