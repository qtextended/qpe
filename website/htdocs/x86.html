Installing Qt Palmtop on Linux/x86

There is no x86 binary demo version of Qt Palmtop 1.4.0, however the previous version, 1.3.1 still makes an excellent demonstration.

The Linux/x86 demo works both on the Linux framebuffer and under X11 (using the Qt Virtual Framebuffer).

  • Download qpe-1.3.1-bin-x86.tar.gz and unpack it.
  • Change to the qpe-1.3.1-bin-x86 directory.
  • Execute the run_demo script.
If you are running X11 then the Qt Virtual Framebuffer will appear. This is just a dumb frame buffer. Qt/Embedded handles all drawing, window management, etc.

If you have the Linux framebuffer configured (16 or 32 bits per pixel) and you start the demo from the console, the Qt Palmtop will use the framebuffer.
