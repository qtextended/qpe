The main() function

The main() function of all applications should following this example: 

int main( int argc, char **argv )
{
    QPEApplication a( argc, argv );
    
    Main m;
    a.showMainWidget(&m);

    a.exec();
}

If the application is document-oriented, the call to QPEApplication::showMainWidget() should be replaced by a cal to QPEApplication::showMainDocumentWidget() .
