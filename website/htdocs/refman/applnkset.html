Detailed Description

The AppLnkSet class is a set of AppLnk objects.

Member Function Documentation

AppLnkSet::AppLnkSet ()

Constructs an empty AppLnkSet.

AppLnkSet::AppLnkSet ( const QString & directory )

Constructs an AppLnkSet that contains AppLnk objects representing all the files in a directory (recursively).

The directories may contain ".directory" files which overrides any AppLnk::type() value of AppLnk objects found in the directory. This allows simple localization of application types.

AppLnkSet::~AppLnkSet ()

Destroys the set, deleting all AppLnk objects it contains.

See also detachChildren().
